# Roadmap

This repository documents a public-safe recommendation architecture for WooCommerce product pages.

## Near Term

- Add small fixtures for signature fingerprints and scoring behavior.
- Add cache-age diagnostics to the documentation.
- Add a sample rebuild command for candidate snapshots.
- Document fallback behavior when candidates are unavailable.

## Later

- Add synthetic before/after render screenshots.
- Add a public ADR comparing signature scoring with heavier recommendation approaches.
- Add a small benchmark note for candidate set limits.

## Not Planned

- Publishing private scoring weights.
- Publishing catalog-specific merchandising rules.
- Publishing product IDs or real recommendation logs.
